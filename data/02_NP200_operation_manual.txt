第1章　装置概要と安全上の注意
================================================================================

1.1 NeuroPulse NP-200 とは
----------------------------------------------------------------------
NeuroPulse NP-200 は、生体神経信号の波形特性を模した微小電気パルスを
高精度に発生・出力するための研究用装置です。
主な用途として以下が挙げられます。

  (a) 神経・筋電図研究における刺激波形の評価
  (b) インビトロ（体外）での細胞応答特性の解析
  (c) ニューラルインターフェース開発における電極特性評価
  (d) 医療機器メーカーによる刺激装置の比較検証（研究段階）

本装置の主要機能:
  ・独立した 8 チャンネルのパルス出力（CH1〜CH8）
  ・波形プリセット 64 種（ゾルタック社定義の ZPW ファイル形式）
  ・振幅範囲 : 0〜±10 mA（電流源モード）、0〜±20 V（電圧源モード）
  ・パルス幅 : 10 μs〜100 ms（1 μs 分解能）
  ・繰り返し周波数 : 0.1 Hz〜10 kHz

1.2 安全上の重要な注意事項
----------------------------------------------------------------------
【警告】感電・障害を防ぐため、以下を厳守してください。

  (1) 本装置の出力端子に人体を接続することは絶対に禁止します。
  (2) 電源投入前に、出力リード線の絶縁被覆の損傷がないか確認してください。
      損傷が確認された場合は直ちにサービス部門へ連絡してください。
  (3) 接液部品（電極・リード線）の交換作業は、必ず出力を停止し
      電源を切った状態で行ってください。
  (4) 引火性・爆発性ガスの存在する環境での使用は厳禁です。
  (5) 本装置は防水構造を持ちません。液体がかからない場所に設置してください。
  (6) 電磁環境 : 電磁適合性（EMC）試験済みですが、強電磁界環境（MRI 室内等）
      での使用・保管は避けてください。

第2章　各部の名称と機能
================================================================================

2.1 前面パネル
----------------------------------------------------------------------

  ┌────────────────────────────────────────────────────────┐
  │ [電源スイッチ]  [緊急停止ボタン(赤)]                        │
  │ [7インチ タッチパネルディスプレイ]   [チャンネルステータスLED×8] │
  │ [エンコーダノブ×2（振幅/周波数）]                           │
  │ [CH SEL] [WAVE SEL] [OUTPUT ON/OFF] [PRESET LOAD]          │
  └────────────────────────────────────────────────────────┘

  各部の説明:
  ・電源スイッチ           : 主電源のオン／オフ
  ・緊急停止ボタン（赤）   : 押下で全チャンネルの出力を即時停止（要手動解除）
  ・タッチパネル           : 波形モニタ・パラメータ設定・ログ閲覧に使用
  ・チャンネルステータス LED :
      消灯  … チャンネル未使用
      緑点灯 … 出力準備完了
      緑点滅 … 出力中
      赤点灯 … アラーム発生
  ・エンコーダノブ右（振幅）: 時計回りで振幅増加、反時計回りで减少
  ・エンコーダノブ左（周波数）: 同上・周波数操作

2.2 背面パネル
----------------------------------------------------------------------
  ・CH1〜CH8 出力コネクタ  : SHV コネクタ（耐高電圧型）
  ・外部トリガ入力         : BNC、TTL レベル、立ち上がりエッジ検出
  ・外部同期出力           : BNC、50 Ω終端
  ・LAN ポート             : RJ-45（1000BASE-T、ZoltaRemote ソフト接続用）
  ・USB-A × 2             : ZPW ファイルのインポート・ログエクスポート
  ・アース端子             : 必ず専用アース線（付属品）で接地すること
  ・電源インレット         : IEC 60320 C14（AC 100〜240 V 自動切換）

第3章　操作手順
================================================================================

3.1 起動手順
----------------------------------------------------------------------
STEP 1: 緊急停止ボタンが解除（引き出し状態）であることを確認する。
STEP 2: 接続する負荷（電極・ダミー抵抗等）をすべての CH に接続する。
        ※ 出力端子をオープン（無負荷）状態で出力 ON にしないこと。
STEP 3: 前面パネルの電源スイッチを「I（ON）」側へ回す。
STEP 4: 起動画面（ゾルタックロゴ）が表示され、約 25 秒で自己診断が完了する。
STEP 5: ホーム画面に「SYSTEM READY」が表示されたことを確認する。
        アラームが表示された場合は「付録A アラームコード一覧」を参照すること。

3.2 チャンネルの設定
----------------------------------------------------------------------
（タッチパネル操作）
(1) ホーム画面下部の [CH SEL] タブをタップし、設定対象チャンネルを選択する。
(2) [WAVE SEL] タブで使用する波形種別を選択する。
    選択可能な波形:
      MONOPHASIC  … 単相パルス
      BIPHASIC-S  … 対称二相パルス（刺激実験の標準）
      BIPHASIC-A  … 非対称二相パルス
      BURST       … バーストモード（パルス列）
      CUSTOM      … ZPW ファイルから読み込みした任意波形
(3) [AMPLITUDE] フィールドをタップし、エンコーダノブまたはテンキーで
    振幅値を入力する（電流／電圧モードの切替はこの画面から実施）。
(4) [PULSE WIDTH] フィールドで パルス幅を設定する。
(5) [FREQUENCY] フィールドで繰り返し周波数を設定する。
(6) [APPLY] ボタンをタップして設定を確定する。

3.3 出力の開始・停止
----------------------------------------------------------------------
■ 出力開始
  前面パネルの [OUTPUT ON/OFF] ボタンを押す。
  → 該当チャンネルの LED が緑点滅に変わり、出力が開始される。

■ 出力停止（通常）
  再度 [OUTPUT ON/OFF] ボタンを押す。
  → LED が緑点灯に戻り、出力が停止する。

■ 緊急停止
  緊急停止ボタン（赤）を手で押し込む。
  → 全チャンネルの出力が即時停止し、ブザーが 3 回鳴る。
  → 復帰するには、ボタンを時計回りに回して引き出し、
    タッチパネルの [ALARM RESET] をタップしてからSTEP 3 以降を再実施する。

3.4 プリセットの保存・読み込み
----------------------------------------------------------------------
■ 保存
  (1) 目的の設定が完了した状態でホーム画面の [PRESET SAVE] をタップ。
  (2) プリセット名（最大 20 文字）を入力し [OK] をタップ。
  (3) 内部メモリに最大 200 件まで保存可能。

■ 読み込み
  (1) [PRESET LOAD] ボタンを押す。
  (2) 一覧から目的のプリセット名を選んでタップ。
  (3) [LOAD & APPLY] をタップして即時反映。

3.5 PC からのリモート制御（ZoltaRemote）
----------------------------------------------------------------------
(1) 付属の ZoltaRemote ソフトウェアをホスト PC にインストールする
    （対応 OS: Windows 10/11 64-bit、推奨 RAM 8 GB 以上）。
(2) LAN ケーブルで NP-200 と PC を接続する（直結またはスイッチ経由）。
(3) ZoltaRemote を起動し、[デバイス検索] をクリックする。
    IP アドレスが自動取得される（DHCP 環境）。
    固定 IP の場合はメニューの [設定] → [ネットワーク] から手動入力する。
(4) 接続後、PC 画面上でパラメータ設定・実験シーケンスの組み立てが可能。

================================================================================
第4章　日常点検と清掃
================================================================================

4.1 使用前点検チェックリスト
----------------------------------------------------------------------
  □ 本体に物理的な損傷（亀裂・変形・焦げ跡）がないか
  □ 電源ケーブルの被覆に損傷がないか
  □ 出力リード線のコネクタがしっかり接続されているか
  □ アース線が確実に接続されているか
  □ 前面パネルの緊急停止ボタンが引き出し状態（解除状態）であるか

4.2 清掃
----------------------------------------------------------------------
  ・前面パネルおよび外装ケース: 柔らかい乾布または水を固く絞った布で拭く。
    溶剤（アルコール・アセトン等）は使用禁止。
  ・通気孔: 圧縮空気（0.2 MPa 以下）でほこりを除去する（3 カ月ごと推奨）。
  ・出力コネクタ（SHV）: 接点クリーナ（弱電用）を綿棒に少量つけて清掃する。
    清掃後は完全に乾燥させてから使用すること。
